1-bosqich

Atamalar 📖

Function / Funksiya
Har bir x ga bitta y mos keladi
\(f: x \mapsto y\). Masalan: \(f(x) = 2x + 1\)
Domain / Aniqlanish sohasi
x ning mumkin qiymatlari
Funksiya aniqlanadigan barcha x qiymatlar to'plami.
Range / Qiymatlar sohasi
y ning mumkin qiymatlari
Funksiya qabul qiladigan barcha y qiymatlar to'plami.
Linear function
Chiziqli funksiya
\(f(x) = kx + b\) — grafigi to'g'ri chiziq.
Quadratic function
Kvadratik funksiya
\(f(x) = ax^2 + bx + c\) — grafigi parabola.
Even function / Juft funksiya
\(f(-x) = f(x)\)
Grafik y o'qiga nisbatan simmetrik. Masalan: \(f(x) = x^2\)
Odd function / Toq funksiya
\(f(-x) = -f(x)\)
Grafik koordinatalar boshiga nisbatan simmetrik. Masalan: \(f(x) = x^3\)
Monotone / Monoton
Doim o'sib yoki kamayib boruvchi
O'suvchi: \(x_1 < x_2 \Rightarrow f(x_1) < f(x_2)\)
Inverse function / Teskari funksiya
\(f^{-1}(x)\)
\(f\) ni "orqaga" bajaradigan funksiya.
Composite function
Murakkab funksiya
\((f \circ g)(x) = f(g(x))\)
Vertex / Cho'qqi
Parabolaning eng yuqori/past nuqtasi
\(x_0 = -\frac{b}{2a}\), \(y_0 = f(x_0)\)
Asymptote / Asimptota
Grafik yaqinlashadigan chiziq
\(y = 0\) — \(f(x) = \frac{1}{x}\) uchun gorizontal asimptota.

2-bosqich

Formulalar 📐

Chiziqli funksiya
\[y = kx + b\]
k — qiyalik (slope): k>0 o'suvchi, k<0 kamayuvchi  |  b — y o'qi kesish nuqtasi
Kvadratik funksiya
\[y = ax^2 + bx + c\]
\[x_0 = -\frac{b}{2a},\quad y_0 = f(x_0) = -\frac{D}{4a}\]
a>0: parabola yuqoriga, minimum  |  a<0: pastga, maksimum  |  x₀,y₀ — cho'qqi
Teskari funksiya
\[y = f(x) \Rightarrow x = f^{-1}(y)\]
\[\text{Topish: } y \text{ ni } x \text{ ga, } x \text{ ni } y \text{ ga almashtiring}\]
Grafigi \(y = f(x)\) grafigining \(y = x\) to'g'ri chizig'iga nisbatan aksidir.
Juft va Toq funksiya
\[f(-x) = f(x) \Rightarrow \text{juft}\]
\[f(-x) = -f(x) \Rightarrow \text{toq}\]
Murakkab funksiya
\[(f \circ g)(x) = f(g(x))\]
\[\text{Masalan: } f(x)=x^2,\; g(x)=x+1 \Rightarrow f(g(x)) = (x+1)^2\]

3-bosqich

Yechilgan misollar 🧮

Har bir misol ustiga bosing — bosqichma-bosqich yechimni ko'ring.

1
\(f(x) = 3x - 2\). \(f(4)\) ni toping.
👁 Ko'rish
1
x = 4 ni formulaga qo'ying
\(f(4) = 3 \cdot 4 - 2\)
2
Hisoblang
\(f(4) = 12 - 2 = 10\)
JAVOB:\(f(4) = 10\)
2
\(f(x) = x^2 - 4x + 3\) funksiyasining cho'qqisini toping.
👁 Ko'rish
1
Formuladan foydalaning
\(x_0 = -\frac{b}{2a} = -\frac{-4}{2 \cdot 1} = 2\)
2
y₀ ni toping
\(y_0 = f(2) = 4 - 8 + 3 = -1\)
3
Cho'qqi
\((2,\; -1)\)
a=1>0 bo'lgani uchun bu minimum nuqta.
JAVOB:\((x_0, y_0) = (2,\ -1)\)
3
\(f(x) = 2x + 5\) ning teskari funksiyasini toping.
👁 Ko'rish
1
y = f(x) deb yozamiz
\(y = 2x + 5\)
2
x ni y orqali ifodalang
\(2x = y - 5 \Rightarrow x = \frac{y-5}{2}\)
3
x va y ni almashtiring
\(f^{-1}(x) = \frac{x-5}{2}\)
4
Tekshirish
\(f(f^{-1}(x)) = 2 \cdot \frac{x-5}{2} + 5 = x \checkmark\)
JAVOB:\(f^{-1}(x) = \dfrac{x-5}{2}\)
4
\(f(x) = x^2 + 1\) funksiyasi juft yoki toqligini aniqlang.
👁 Ko'rish
1
f(-x) ni hisoblaymiz
\(f(-x) = (-x)^2 + 1 = x^2 + 1\)
2
f(-x) = f(x) tekshiramiz
\(x^2 + 1 = x^2 + 1 \checkmark\)
3
Xulosa
\(f(-x) = f(x) \Rightarrow \text{funksiya JUFT}\)
Grafigi y o'qiga simmetrik.
JAVOB:Juft funksiya
5
\(f(x) = x^3 - x\) funksiyasi juft yoki toqligini aniqlang.
👁 Ko'rish
1
f(-x) ni hisoblaymiz
\(f(-x) = (-x)^3 - (-x) = -x^3 + x\)
2
-f(x) ni hisoblaymiz
\(-f(x) = -(x^3 - x) = -x^3 + x\)
3
Taqqoslash
\(f(-x) = -f(x) \Rightarrow \text{funksiya TOQ}\)
Grafigi koordinatalar boshiga simmetrik.
JAVOB:Toq funksiya
6
Agar \(f(x) = 2x + 1\) va \(g(x) = x^2\), \(f(g(3))\) ni toping.
👁 Ko'rish
1
Avval g(3) ni hisoblaymiz
\(g(3) = 3^2 = 9\)
2
Keyin f(9) ni hisoblaymiz
\(f(9) = 2 \cdot 9 + 1 = 19\)
3
Natija
\(f(g(3)) = 19\)
JAVOB:\(f(g(3)) = 19\)
7
\(f(x) = \frac{1}{x-2}\) funksiyasining aniqlanish sohasini toping.
👁 Ko'rish
1
Mahraj nolga teng bo'lmasligi kerak
\(x - 2 \neq 0 \Rightarrow x \neq 2\)
2
Aniqlanish sohasi
\(D(f) = (-\infty, 2) \cup (2, +\infty)\)
3
Boshqa yozuv
\(D(f) = \mathbb{R} \setminus \{2\}\)
x = 2 bundan mustasno.
JAVOB:\(x \in \mathbb{R},\; x \neq 2\)
8
\(f(x) = \sqrt{x - 3}\) ning aniqlanish sohasini toping.
👁 Ko'rish
1
Kvadrat ildiz ichidagi ifoda ≥ 0 bo'lishi kerak
\(x - 3 \geq 0\)
2
Yechamiz
\(x \geq 3\)
3
Aniqlanish sohasi
\(D(f) = [3, +\infty)\)
JAVOB:\(x \geq 3\)
9
\(f(x) = x^2 - 6x + 8\) funksiyasini \(a(x-h)^2 + k\) ko'rinishiga keltiring.
👁 Ko'rish
1
To'liq kvadratga ajratamiz
\(x^2 - 6x + 8 = (x^2 - 6x + 9) - 9 + 8\)
9 qo'shib olamiz, keyin ayiramiz.
2
Soddalashtiramiz
\(= (x - 3)^2 - 1\)
3
Cho'qqi
\(h = 3,\; k = -1 \Rightarrow (3, -1)\)
a=1>0 bo'lgani uchun bu minimum.
JAVOB:\(f(x) = (x-3)^2 - 1\)
10
\(f(x) = 2x^2 - 8x + 6\) funksiyasining o'sish va kamayish oraliqlarini toping.
👁 Ko'rish
1
Cho'qqi x koordinatasi
\(x_0 = -\frac{b}{2a} = -\frac{-8}{4} = 2\)
2
a = 2 > 0 — parabola yuqoriga ochiladi
\(\text{Kamayish: } x \in (-\infty, 2)\)
Cho'qqigacha kamayadi.
3
O'sish oralig'i
\(\text{O'sish: } x \in (2, +\infty)\)
Cho'qqidan keyin o'sadi.
4
Minimal qiymat
\(f(2) = 8 - 16 + 6 = -2\)
JAVOB:\(\text{Kamayish: } (-\infty, 2), \text{ O'sish: } (2, +\infty)\)
