❓ Web Development Questions

Click a question to reveal the answer

1. What is the difference between frontend and backend development?
Frontend focuses on the user interface and design (HTML, CSS, JS), while backend handles server logic, databases, and application functionality.
2. Why is semantic HTML important?
Semantic HTML improves accessibility, SEO, and makes code more structured and meaningful for both browsers and developers.
3. What is responsive design?
Responsive design ensures websites adapt to different screen sizes and devices using CSS media queries and flexible layouts.
4. What is an API?
API (Application Programming Interface) allows different software systems to communicate and exchange data with each other.
5. Why is version control important?
Version control (like Git) helps track code changes, collaborate with teams, revert mistakes, and manage the entire code history.
6. What is the difference between a class and an ID in CSS?
An ID (#) is unique and can only be used once per page. A class (.) can be reused on multiple elements. IDs have higher specificity than classes.
7. What does the DOM stand for and what is it?
DOM stands for Document Object Model. It is a programming interface that represents the HTML document as a tree of objects that JavaScript can interact with.
8. What is the difference between == and === in JavaScript?
== checks for equality with type coercion (e.g., "5" == 5 is true). === checks for strict equality without coercion (e.g., "5" === 5 is false). Always prefer ===.
9. What is CSS Flexbox used for?
Flexbox is a CSS layout model that makes it easy to align and distribute elements in a container, especially in one direction (row or column).
10. What is the purpose of localStorage in JavaScript?
localStorage allows you to store key-value data in the browser that persists even after the page is closed. It is useful for saving user preferences or session data.
